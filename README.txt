Fixture Sheet Generator

Overview

This script, Fixture Sheet Generator, is a tool designed to streamline the process of creating lighting fixture sheets.

Features

- Generates a comprehensive fixture sheet from user input.
- Supports multiple fixture types and custom attributes.
- Exports data to a structured format for easy reference.

Requirements

- Python 3.x
- The following Python libraries (can be installed via pip):
  - pandas
  - openpyxl (for Excel support)
  - xlsxwriter (required for Excel export)

Installation

1. Clone the repository:
   git clone \<repository\_url>

2. Install the required libraries individually by running the following commands:

   pip install pandas

   pip install openpyxl

   pip install xlsxwriter

Usage

1. Run the script:
   python Fixture Sheet Generator.py

2. Follow the prompts to input the required fixture details. The script will ask for information such as fixture type, position, channel, and more.

3. Once completed, the script will generate a fixture sheet in Excel format for easy use. The file will be downloaded to the same directory where the script is run.
